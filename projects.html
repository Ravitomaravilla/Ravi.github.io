Blue Horizontal Branch (BHB) Stars

During my first real experience with scientific research and observations, I worked on a project focused on the spectroscopic analysis of Blue Horizontal Branch (BHB) stars to investigate their evolutionary origin from main-sequence stars without requiring stellar interactions. The aim was to acquire multi-epoch spectra of BHB candidates in order to identify radial velocity variations caused by potential binary companions and to compare binary frequencies with those of main-sequence populations.

This involved target selection using tools such as TOPCAT for catalog and target list creation, Simbad for object identification, and Stellarium for generating finding charts. Once observations were obtained, I performed reduction and preliminary checks on the spectra, including line broadening analysis to study stellar rotation and other broadening mechanisms, as well as SED fitting to determine stellar parameters.

I also explored the use of IRAF for data reduction and spectral analysis. This project not only deepened my understanding of stellar astrophysics but also gave me hands-on exposure to observational planning, data reduction, and interpretationâ€”an inspiring introduction to doing real science.

Classification of Galaxies

This project focuses on the morphological classification of galaxies using convolutional neural networks (CNNs). The goal was to train a deep learning model capable of distinguishing between spiral and elliptical galaxies. The dataset was obtained from the Galaxy Zoo project, with FITS file extraction and data cleaning performed using Astropy and Pandas, followed by filtering uncertain classifications and downloading images via the SDSS API using the Requests library.

I preprocessed and augmented the dataset, normalized the images, and trained a CNN using TensorFlow and Keras. All images were standardized with Pillow and NumPy, then organized into structured training and validation sets. The model learned key morphological patterns such as spiral arms and bulges, achieving approximately 95% validation accuracy.

The final model demonstrated strong performance in predicting galaxy morphologies, showcasing the power of machine learning in astrophysical data analysis.
